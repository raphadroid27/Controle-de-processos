Visão Geral

O Controle de Processos é um aplicativo desktop para registrar, acompanhar e analisar pedidos/propostas, facilitando o gerenciamento de informações e indicadores.

Fluxo de trabalho

  1. Cadastrar um novo registro informando Cliente, Pedido/Proposta, Quantidade de Itens, Data de Entrada, Data de Processo, Tempo de Corte e Valor.
  2. Filtrar por cliente, pedido, ano e período de faturamento (26 → 25) para análises específicas. Administradores podem filtrar por usuário.
  3. Consultar Totais para métricas agregadas (itens, valor, médias) e o Dashboard (somente administradores) para análises por período e usuário.

Conceitos-chave

  • Período de faturamento: do dia 26 de um mês ao dia 25 do mês seguinte (ex.: 26/03 → 25/04). Toda análise mensal segue esse recorte.
  • Tempo de corte: duração em HH:MM:SS do corte a laser; opcional, mas essencial para indicadores de horas diárias.
  • Data de Entrada vs. Data de Processo: a primeira marca quando o pedido chegou; a segunda, quando foi concluído. A diferença entre elas calcula os "dias de processo". Registros sem "Data de Processo" não entram no cálculo da média de dias.
  • Dias úteis: o sistema considera apenas dias úteis (segunda a sexta) para calcular médias diárias e estimativas mensais, excluindo finais de semana automaticamente.
  • Edição in-line: clique duas vezes em qualquer célula da tabela para editar diretamente. Campos de data abrem um calendário; valores numéricos validam o formato automaticamente.
  • Filtros combinados: todos os filtros (Cliente, Pedido, Ano, Período, Usuário) funcionam em conjunto. Os totais refletem sempre o resultado filtrado.

Interface principal

A janela é dividida em três áreas verticais:

  1. Formulário de entrada: campos para cadastrar novos registros (Cliente, Pedido/Proposta, Quantidade de Itens, Data de Entrada, Data de Processo, Tempo de Corte e Valor). O campo Cliente possui autocompletar. Observação: os campos de data são atualizados automaticamente para a data atual quando o aplicativo é iniciado ou quando o dia muda.
  2. Filtros e Tabela: painel de filtros (Cliente, Pedido, Ano, Período e Usuário se for admin) e tabela com edição in-line. Datas abrem calendário ao clicar duas vezes; exclusão solicita confirmação.
  3. Painel de Totais: métricas agregadas (pedidos, itens, valores, médias) sempre sincronizadas com os filtros ativos.

Menus

  • Arquivo: Logout (Ctrl+Shift+L) e Sair (Ctrl+Q).
  • Admin (somente administradores): Ferramenta Administrativa (Ctrl+G), Dashboard (Ctrl+D), Atualizar tabela (F5).
  • Tema: alternar entre Claro/Escuro, escolher estilo (Fusion/Windows) e cor de destaque.
  • Ajuda: Manual do Sistema (F1) e Sobre.

Barra de status: exibe o nome do usuário logado e se possui permissões de administrador.

Atalhos de teclado

  • Ctrl+Q: Sair da aplicação (fecha o sistema completamente).
  • Ctrl+Shift+L: Fazer logout (retorna à tela de login sem fechar o sistema).
  • F1: Abrir o manual do sistema.
  • F5: Atualizar tabela (somente administradores).
  • Ctrl+G: Ferramenta Administrativa (somente administradores).
  • Ctrl+D: Dashboard (somente administradores).
