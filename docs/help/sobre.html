Sobre a Aplicação

Controle de Pedidos — sistema desktop para gestão de pedidos e análise de indicadores operacionais.

Informações da versão

  • Acesse Ajuda → Sobre no menu principal para visualizar a versão atual, autor e link do repositório.
  • A janela "Sobre" exibe: nome da aplicação, versão, autor (raphadroid27), descrição e link para o GitHub.

Tecnologias

  • Interface: PySide6 (Qt for Python)
  • Banco de dados: SQLite
  • Gráficos e análises: Matplotlib e Pandas

Suporte

  • Para dúvidas sobre uso, consulte este manual pressionando F1.
  • Para problemas técnicos ou sugestões, contate o administrador do sistema.
  • O código-fonte está disponível no repositório GitHub.
